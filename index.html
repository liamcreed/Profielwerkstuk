Welke muzikaal aspect beinvloed een film scene het meeste?

Hypothese

Ik verwachtt dat toonsoort een film scene het meest kan beinvloeden.

Onderzoek

Ik heb 3 muzikale elementen gekozen: toonsoort, timbre en tempo. Voor elk element heb ik een filmscene gekozen en twee door het gekozen element contrasterende muziekstukken gemaakt. Daarna ga ik een paar mensen vragen om met behulp van een emotie schaal te onderzoeken welk muzikaal element het grootste verschil maakt.

Muziekstukken Toonsoort

Ik ben een grote fan van Vangelis. Voor een geluidsfragment wilde ik een stuk van hem namaken. Voordat ik uit was over welke compositie ik ging kiezen, had ik als test het thema van Halloween (John Carpenter) nagemaakt en de toonsoort veranderd. Hierdoor had ik ontdekt dat ik een andere kant op wilde gaan met mijn profielwerkstuk en vooral wilde gaan focussen op het maken van muziek en daarmee onderzoek doen in plaats van veel bronnen lezen. Uiteindelijk heb ik het thema van chariots of fire gekozen.

Halloween (1978) thema mineur
Halloween (1978) thema majeur

Ik heb het programma logic pro gebruikt. Logic pro heeft genoeg instrumenten om alle partijen na te maken.

Logic Pro

Eerst heb ik de piano partij geleerd. Ik heb thuis geen akoestische piano dus heb ik een midi keyboard gebruikt met een piano plugin om deze partij op te nemen. Dit was handig omdat ik later nog het type piano kon aanpassen.

Piano plugin Keyscape

Daarna heb ik de percussie nagebootst. De percussie heb ik gemaakt met witte ruis met laagdoorlaatfilter dat oscileert waardoor er een pulse ontstaat. De synth bass speelt alleen een c# in 16e noten. Voor de bass heb ik de Logic pro retro synth gebruikt.

Logic Pro Retro Synth

De strijkers spelen samen een akkoord. Het akkoord wisselt af van een C# naar een F#. Voor de strijkers heb ik ook de Retro Synth gebruikt met de preset bright strings

Voor de synth blazers gebruikte Vangelis zelf de Yamaha CS80. Deze synthesizer heb ik helaas niet (kost ook maar een kleine €70.000). Gelukkig kreeg ik met mijn midi keyboard gratis software met een paar geluiden van de CS80. Deze heb ik gebruikt voor de blazers.

Vangelis met de Yamaha CS80

De compositie staat origineel in majeur dus nadat alle partijen af waren heb ik alle partijen omgezet naar de mineur toonsoort. Omdat ik alle had opgenomen als MIDI hoefde ik niets opnieuw te spelen.

Majeur veranderen in Mineur

Ten slotte heb ik het gemixed, geexporteerd en daarna het beeld er bij geplakt.

Geluid onder video plaatsen in Davinci Resolve

Het was erg leuk om de compositie en geluiden na te maken. Het was lastig om uit te vogelen wat de instrumenten speelden omdat ik alleen naar de muziek heb geluisterd. Ik ben erg tevreden over het resultaat.
